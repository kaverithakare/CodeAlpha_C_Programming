CodeAlpha C Programming Internship

📌 About

This repository contains the C Programming projects completed during my CodeAlpha C Programming Internship. The purpose of these projects is to strengthen my understanding of C programming concepts through practical implementation. Each task focuses on different programming concepts such as arithmetic operations, functions, arrays, structures, file handling, and modular programming.

---

🚀 Project Tasks

📖 Task 1: Basic Calculator Program

This project is a menu-driven calculator developed in the C programming language. It allows users to perform basic arithmetic operations using the Switch Case statement.

Features:

- Addition
- Subtraction
- Multiplication
- Division
- User-friendly menu
- Option to perform multiple calculations

Concepts Used:

- Switch Case
- Functions
- User Input and Output
- Conditional Statements

---

📖 Task 2: Matrix Operations

This project performs various matrix operations using Functions and Two-Dimensional Arrays. The program provides a menu for selecting different matrix operations.

Features:

- Matrix Addition
- Matrix Multiplication
- Transpose of Matrix
- Modular programming using functions

Concepts Used:

- 2D Arrays
- Functions
- Nested Loops
- Matrix Manipulation

---

📖 Task 3: Student Management System

This is a menu-driven Student Management System developed using Structures and File Handling in C. The program stores student records permanently in a file and allows users to manage them efficiently.

Features:

- Add Student Record
- Display All Student Records
- Search Student Record
- Update Student Record
- Delete Student Record
- Permanent Storage using File Handling

Concepts Used:

- Structures
- File Handling
- Functions
- Menu-Driven Programming
